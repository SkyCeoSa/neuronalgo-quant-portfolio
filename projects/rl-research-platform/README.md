# RL Research Platform (Work-in-Progress)

این پروژه، محیط تحقیقاتی برای آزمایش عامل‌های تقویتی (RL) همراه با کامپوننت‌های ML است.

هدف‌ها:
- ساخت محیط بازار ساده (simulator) برای آموزش agent
- پیاده‌سازی چند الگوریتم پایه (DQN, PPO)
- افزودن reward shaping و risk-aware reward
- ارزیابی agent با معیارهای مالی (CAGR, Sharpe, MaxDD)

این فولدر شامل:
- README.md (همین فایل)
- roadmap.md
- example_agent.py (اسکریپت نمونه)
- notebook_instructions.md
